　　#

　　从朝歌出发至冀州，需行四百余里过千城，越往北边走，寒气越盛，风里开始夹着雪。将至冀州，忽见一处未被风雪侵袭，有一城门大小裂缝直通山体内里，引得探路的三人探入，别有洞天。误入瑶池，算是意外之喜。

　　“什么瑶池，就一眼破温泉。”

　　崇应彪扯了扯缰绳，水面袅袅热烟逼得胯下悍马后退几步，他轻轻瞥一眼殷郊，后者并没有在意，他在下意识说出“瑶池”二字后明显怔住，只急急下马，绕池一圈细细打量这个地方，连一草一石都要看清纹理来。崇应彪又看眼姬发，果不其然，他的目光只在粗略环视一番周边后，便跟随着殷郊。

　　冰冷盔甲御敌不御寒，狐裘披风在咧咧寒风中只能挡下部分，而更细碎狡猾的余风总能趁乱钻入殷商勇士们的里衣，再将暖意窃取，给勇士们留下的颤栗。在这般环境，偶然发现这汪冒着热气的热泉，说不是瑶池，但胜于瑶池。是个休整的好地方。

　　“你们稍作歇息，我回去禀报主帅，”见殷郊正望着水面发愣，姬发拍拍马背说道，回应他的是马儿不忿般的甩头，“我带他们过来。”

　　“别，我去，我不忍拆开你们。”

　　“你又打什么注意。”

　　“就这么看我？我真心成全。”

　　姬发抬头，带着愠怒的目光瞪向崇应彪，崇应彪回他轻蔑的笑，脚下一踢马腹，拉高缰绳就此离开。姬发不再看马蹄扬起的烟尘，牵马到一旁拴牢，便卸下盔甲，叠整齐放好在身旁，偌大的空间过于寂静，殷郊还是一言不发。

　　“殷郊，殷郊，殷郊！”

　　殷郊从沉思中回神，望着姬发询问的眼，摆了摆手表示无事，取下厚重铜盔，边卸甲边说道：

　　“只是在想，如果我们先一步泡上这温泉，对弟兄们是不是不公平。”

　　“想得倒是周全，全营几千弟兄，同时都挤在一个池子里下饺子？”

　　“要不就等到文焕全孝他们来。”

　　“如果留在这里的是崇应彪，怕是已经在这里游三圈了。”

　　“你们刚刚在吵什么？”

　　和崇应彪发生争吵简直和呼吸一样轻易，偏偏每次殷郊都问缘由，太鸡毛蒜皮的事姬发也不是都想回答，包括这次。姬发背对着殷郊，解开衣带褪下里衣：“你确定要等他们，我可不……”

　　话未说完，身后便是落水的声音，姬发回头，殷郊已经跳入泉中，热泉刚好没过胸口，面容在雾气中变得模糊，但他笑意盎然，姬发看得真切，自知又被这幼稚的伎俩摆了一道，只无奈摇摇头，迅速脱下剩余的衣物，一并跳入水中。

　　热流将姬发包围，逐渐驱赶寒气，他情不自禁地呼出一口气，准备解下缠在手间的绑带，一道灼灼目光打在他的背上，他顿时心跳如擂鼓。

　　风雪中罕有的遮风秘地，在此美好幽境，四下无人，他们心照不宣地任无旁人知晓的情愫与悸动随着这汪热泉荡漾开。姬发想起，距离那夜已有三月。

　　三月的时间不远，只是情生情起，得从头娓娓道来。

　　
　　#　

　　姬发初来朝歌时，曾对贵为皇子的殷郊与八百诸侯质子混在质子营感到不解，和其他同期入营的质子们一样，姬发都当殷郊是公子哥，仗着父亲是将领的身份来体验兵营生活，来当个轻松的巡视官，遛一圈便回去。

　　直到他手中的剑被殷郊挑下。

　　年幼尊贵的皇子身穿与他一致的劲装，头发紧束，刚刚结束的对打并不简单，殷郊气喘如牛，嘴角因胜利而扬起，灿如骄阳。这是他们第一次交锋，姬发从打斗中殷郊的眼神知晓，眼前的人并不是玩玩而已，与他一样是卯足了力气去赢。

　　殷郊赢了，从那时起姬发正式将殷郊列入竞争对手行列，对抗训练时，姬发特意上前邀请殷郊作为搭档，也只有对抗训练的时候是，当要求分队攻防战时，姬发永远站在殷郊对面。一来二去的，殷郊知道姬发抱的什么心思，但他没有为此反感，毕竟他的身份本就容易招来事端，只有他敢说他绝对是众多质子的假想敌首选。

　　来去质子这么多，殷郊总会多看姬发一眼，他记得西岐以农耕为主，但这位来自西岐的少主，除了原生的质朴纯粹之余，多几分专注、倔强和狠劲，倒是像丛林里的幼狼，想和自己争山头。殷郊欣然接下所有挑战，是想有天这匹小狼能明白，这山头还不是他殷郊的，在真正相争之前，他们其实可以站在同一侧。

　　在姬发第二次充当“主帅”的攻防战中，殷郊败了，还是惨败。先是其中一名“小将”错误判断，贸然出动，打乱了整局的战术，姬发抓住时机，势如破竹，殷郊补救不及，又在保卒还是迎战之间犹豫，姬发已经奔至他眼前，几个来回竟然被挑到马下，殷郊仓皇回头，眼看着他方战旗被夺，便知他与这夜的庆功酒无缘。

　　当夜，这厢篝火烈烈，谈笑欢饮，另厢的殷郊在主帅营房领了十五下鞭罚。殷寿没有丝毫犹豫，也没有收半分力气，他看着跪在面前的亲生骨肉原本结实有力的赤裸后背，在重鞭之下盛开血花，眼中不悲不喜。殷郊由始至终未发出声响。

　　到底是年少，殷郊忍得大汗淋漓，而汗珠又落入新伤，痛上加痛，一出营房便把手撑在墙面，缓了半天才把气理顺，身形有些摇晃，慢慢走着，背上的伤痛得厉害，而心有浓云，更是如何都挥不开。

　　殷郊不敢回寝殿，生怕被母亲撞见，打算去姜文焕的营房处理好伤口，将就一晚，第二日伤口不渗血，再去拜访母亲也不会露陷。之前是由姜文焕领着他来过，现在他只能凭个大概记忆，掀了其中一个营房的幕帘进去。

　　见四处无人，殷郊便开始自行找金创药——他的背只敢用热水湿巾简单擦擦，沾到一点水都让殷郊痛得头皮发麻，不上药怕是真的会溃烂出脓。

　　姬发刚洗漱回来，左手撩开幕帘，右手捧着面盆，就见一个背上血痕累累的人在他房中翻箱倒柜，尚未发问，那人已经转身，四目相对，唯有烛影跳动。殷郊也没想到来人是姬发，相比较战败者遇上胜利者的局促，殷郊更为他来错营房感到尴尬，沉默半响，先说声“打扰”，便想直接离开这里，二人擦肩而过时，姬发抓住殷郊手臂。

　　“我替你上药。”

　　话说完便松开手，姬发把去或留的选择还给殷郊，也料定殷郊会留下，自顾自地去把药瓶取来。果不其然，殷郊回身落座，姬发向他走来，他的眼一直看着姬发，直至他们对视，殷郊开口，才发现嘴巴异常苦涩，原来从受罚开始心中的苦已经蔓延至喉头：

　　“谢谢。”

　　姬发领了谢，他垂眸，远看殷郊的背他还没有实感，近看才看清原来鞭出的伤痕也能这么重，姬发手上蘸了药酒的棉团找不到合适的落处，左右端详着，发现有些地方伤得深，是因都落在了同一处，姬发不敢细想这是有意还是无意，深如沟壑的伤口微微刺痛他的眼。

　　殷郊也很折磨，背上的痛固然在，等姬发有所动作更令他难受，他看不见姬发的神情，每当后背一阵凉意靠近，他以为终于要下手，把心理建设做了个遍，准备迎接药酒入肉的刺痛，但痛觉又迟迟不来，便缓缓松懈下来，如此反复几回，在殷郊最不提防的时候，姬发的棉团压在了伤痕最浅的地方。

　　“啊！”

　　闻声姬发眉头一跳，他看着眼前的背肌剧烈颤抖了一瞬，他动作不停，只手下力度轻了几分，同时又想，这个耐受力是如何挺过主帅十五鞭的。这可冤了在受罚时险些咬碎牙的殷郊，但方才的声响确实太大，殷郊自觉丢人，他肘撑桌半握拳，前牙轻咬指节，以防再叫出声时能及时堵住嘴。

　　现下营房内全是殷郊吸气声，还有被强硬推回肚子里的闷哼，血污很快就脏了白棉，姬发换得频，不多时他们脚下已堆满深红色，像极秋天跌落在马道上的木棉。

　　“只罚你一人？”

　　姬发缠好了最后一圈纱布，不经意问道，细看渗血情况不严重，便开始收拾，殷郊也想帮忙拾起血棉，奈何他稍微一动便觉得整一片背牵连着痛，只好作罢。

　　“‘兵败，将领罚’，这是规则。”

　　“你没与主帅说这场战中有谁出错？一个都没说？”

　　“……”殷郊沉默，姬发撇他一眼，要说也是竞争关系，但姬发止不住多说几句。

　　“你受罚了是你长记性，以后真在战场，他若再犯，你替他命吗？”姬发见人还是不语，他为说不通殷郊感到恼火，不知道哪里来的勇气，他笃定殷郊所想，索性直接说破，“现在就想当我们统领？主帅对我们都一视同仁。”

　　“那他为何，罚我永远是你们的双倍？”

　　姬发哽住，他几乎瞬间就接收到了殷郊的情绪，他只是没想到殷郊会对他说出这样的话，这算是他们第一次正式的交流，也是第一次，就听到殷郊一直郁郁寡欢的缘由。姬发不觉得殷郊的话突兀，也不为窥看到皇子脆弱一面的升起虚荣感，反而觉得殷郊早该说出口了，作为搭档，他们其实已经足够熟悉对方，一举一动了然于心，而这句打开心扉，是如此自然，你说着，我听着。

　　姬发原以为他们一样，仰慕着强悍威武的英雄，并想着有天能成为他的模样，目的一致，所想便一致。今日才知，殷郊的心与步伐，较他沉重得多，姬发声音柔和了些：

　　“以前我不知，不过这次……不要怪主帅生气，他罚重你，是觉得你在包庇下属，这是大忌。我相信他没有别的意思。”

　　“是吗？”

　　“是。”

　　有没有接受姬发对殷寿的辩解只有殷郊知道，他站了起来，试着走两步。药酒有麻醉功效，被包扎好的背已经没有来时疼痛，身体开始酸软，疲惫感凶猛地袭来，殷郊看向姬发：

　　“我今晚可以在你这里睡一觉吗？”

　　姬发望着殷郊疲倦的眼，他没有问殷郊为什么不回自己的寝殿，而是做了个请的手势：“你不嫌弃就好。”

　　殷郊摇摇头，得了允许的他倒是不客气，径直就往床走去。他们都累极了，尤其是殷郊，他需要任何事物来接住他的身体，或者他的灵魂，让梦仙抽掉脑子里乱七八糟的思绪，然后一觉天亮，继续为成为父亲最满意的儿子拼尽全力。

　　待姬发收拾完毕，殷郊已经面朝里睡下，留了大半空位给他，姬发将薄被踢到床尾，空间更加之大。他直直躺下，细听着身旁殷郊的呼吸，很轻，没睡或已是浅眠。
　　
　　“殷郊，”姬发头枕在手臂上，看着顶蓬花纹，“你下次要是不方便回殿，都可以留在我这里。”

　　良久没有得到回应，姬发半支起身子，小心避开殷郊的背，探头去看殷郊的脸，后者双目紧闭，浓密睫毛快要完全盖过眼底卧蚕，姫发当他已完全睡去，也不再说话，转身抬手挥灭床头烛火。

　　黑夜覆盖二人，困意接踵而来，在姬发闭上眼前，殷郊的声音传来，声音不大不小，正好落入姬发耳中。

　　“好。”

　　黑暗中姫发眨眨眼，而后阖上。

　　今夜好眠无梦。

　　春去冬来又几年，质子们脸上的稚嫩逐渐褪去，殷寿操练下的他们体格健壮，气宇昂扬，战甲一戴，好不英姿飒爽，刀剑枪棍无所不能，俨然已是一群少年将军。最为出众者，当属殷郊与姬发，彼时他们已是同出同入，虽训练场上依然拔刃张弩，但退场后便谈笑风生，任谁看了都知道他们关系密切。

　　他们形影不离到连宫中姜王妃都知晓。

　　秋祭前日，姬发脚刚踏入庭院，便被殷郊一把拽了去，被按坐在一把石椅上，殷郊返回屋里，拿来一盏茶，又再次匆匆走开。姬发看着殷郊殿中庭内来回奔走，冒冒失失的像个幼儿，无声笑笑，他支着下巴抬头看一树桂花，等候殷郊和他说的惊喜。这里是姜王妃的后院。

　　待在这里的殷郊最是轻松，没有任何拘束，殷郊的笑容也是最多的，他的少年心性尽数存放在这个圣地，在最能任性的人面前，没有行事对错，在父亲前时刻紧绷着的弦，由母亲的安抚下得以回归柔软。姬发本不能来的，但是他是以殷郊好朋友的名义来到这里。

　　姜王妃兰质蕙心，温婉恬静，每每他来也不会生分，抚琴闲谈，听书传道，有殷郊三分疼惜，便有姬发一分关怀。　　

　　她心疼殷郊，并不会因此阻止殷郊，她知孩儿志向，唯有忧心难以排解。而在看到被儿子带到她面前来的西伯侯之子后，她知她的忧已减半，西伯侯为人仁厚，爱民如子远近闻名，他的儿子便是在这般家风下长成，起码大道仁义在心，多几次接触后，更是验证她的想法。

　　“你比郊儿沉稳，在他莽撞之时，拉他一把，他会听你的。”

　　姬发记着这句话，因为他认同这句话，但是拉一把也得看情况，毕竟年少气盛，比如和崇应彪的架，不打是不行的。

　　殷郊带着风在他面前坐下，姬发眼疾手快抓住了一朵将要落入杯中的花，见眼前不大的石板桌上已放好一把古琴，姬发看着殷郊，长发半束斜搭在左肩，柔和他日渐锋利的下颔，薄唇藏不住笑意，剑眉劈开山峦，暖阳下双眸盛着秋水，正望着自己。姬发知他有话要说，但这不妨碍姬发看着平日一拳能打得飞沙走石的手，轻柔地停在五弦琴上时，无端联想：

　　这双手，如果一直抚琴，也是极好的。

　　“你可听好，我学了有些时候。”殷郊收起盈盈笑眼正色道，姬发便收起手，端正坐好，定定看着面前人。

　　殷郊起手，指跪五徽，淌撞往来，勾剔拨刺，如珠落玉盘清脆，同声应合，雨落芭蕉声纷至沓来，少息，缓急有序，再作。

　　姬发听着，耳边弦乐带着他思绪游离远方，带他去白雾缥缈的仙山，光明永照的山顶只有两人，一人抚琴，一人舞剑，抚琴的人换成了他，而树下一身素衣舞剑的人，姬发目光追着望去，那人随剑法纷飞的衣袖始终遮住了脸，唯有行云流水的剑穿云破雾，剑身返照日光，由头到尾姬发满眼金灿。五弦共震，沉而悠远，姬发忽得清明，脑中幻象逐渐与眼前的人重合，是殷郊。姬发知道殷郊弹的是什么曲子。

　　《凤求》，用以对爱慕之人表达深情之意。

　　姬发从前并不对这种过于文雅的事情感兴趣，他只想当个舞刀弄剑的大英雄，琴棋书画不能在战场上帮助他分毫，但他有个文武双全的兄长，未及束发便学会整本琴谱。一日他好事心起，要考考哥哥，证实是真是假。

　　幼年姬发字未识全，随手便指了一曲，伯邑考莞尔，十指似蝶在琴弦上翩然起舞，高山流水自指间淌过，音音细韵，姬发不明音律，但也听痴了，曲终仍浑然不觉，想着让长兄再来一次，伯邑考轻摇头，他问姬发：

　　“这首曲子，听出什么来了？”

　　“唔……叮叮咚咚，好听。”

　　伯邑考笑着看姬发憋了半天才说出这么些评价，将食盘中的一小块桂花糕送入他口中，似水眼眸注视着未知世事的胞弟。

　　“早晚会是我的。”

　　“什么？”殷郊停了话题，姬发的话和他正在说的完全没有关联。

　　“鬼侯剑。”

　　殷郊想了想，也不气姬发完全没听他说话，而是笑开来，笑得星目化为弯月，笑得郎朗清风里都是他的快意。

　　“那你得赢我。”

　　“我能赢得你。”

　　殷郊挑眉，他见姬发并未意识到话中藏有歧义，眼神坚毅，一副势在必得的模样，便不再回应。

　　“我有一事，想问你许久。”

　　姬发喉间滚出一声应答，他摇晃着腿，脚后跟轻碰在古老坚固的墙砖上，他想不起自己身上有什么还能被殷郊问起，西岐的风、西岐的月、西岐的一切，殷郊未见过未听闻的，都已尽数用言语为他补全。

　　“难得，倒是这个时候你有颗玲珑心，”姬发讪笑，他心中挣扎不已，索性顺着殷郊的话头，半开玩笑的语气或许能让他们的关系从悬崖边上拉回，“你是在回味时发现的？”
